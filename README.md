# my-aurelia-app

### Description
Aurelia App - exploring aurelia.js

### Prerequisites
NodeJS

### Installation
To clone

```git clone https://github.com/karinlj/myAureliaApp.git```

To install

`npm install` or `npm install -g aurelia-cli`

To run

`au run`, then open `http://localhost:8080`

### Built With
* aurelia.js
* Less
* Bootstrap 4
